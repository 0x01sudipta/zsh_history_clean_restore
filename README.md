# zsh_history_clean_restore
This script is to help you hide and restore your command history when needed.

# Usage
# Clone Repository
    $ git clone https://github.com/0x01sudipta/zsh_history_clean_restore.git
# Change Directory
    $ cd zsh-history-cleaner
# Clean History
    $ ./clean.zsh
# Restore History
    $ ./restore.zsh
